HTML entities:

some characters are reserved in HTML if you use the less than (>) or greater than (>) signs in your text, the browser might mix them with tags. Character entities are used to display reserved characters in HTML.

1.non-breaking space

A commonly used entity in HTML is the non-breaking space &-nbsp; a non-breaking space in a space that will not break into a new line two words separated by a non-breaking space will stick together (not break into a new line). This is handy when breaking the words might be disruptive. For example (H 2  O)

2.html symbol intities

Many mathematical technical and currency symbols, are not present on a normal keyboard to add with symbol to an HTML page. You can use an HTML entity. If no entity name exists you can use an entity number a decimal or a hexadecimal reference.
